# Пинг-монитор (Лабораторная работа №3 — CI/CD)

Java-утилита для мониторинга доступности хостов через ICMP ping с красивым отчётом.

## Особенности
- Чтение списка хостов из файла
- Поддержка Windows и Linux (парсинг разного вывода ping)
- Статусы: UP / FLAKY / UNREACHABLE
- Сортировка по статусу и RTT
- Покрытие тестами > 70% (JaCoCo)
- Fat-jar (один файл для запуска)
- CI/CD на GitHub Actions (JDK 11/17/21)
